🛒 E-COMMERCE-SALES-PERFORMANCE-ANALYSIS

📌 Purpose of The Project

The purpose of this project is to analyze e-commerce sales data to understand customer purchasing behavior, sales performance, and product trends.
To identify sales patterns and key drivers by comparing store-wise, category-wise, and year-wise sales trends.
The project helps businesses make data-driven decisions to improve revenue, customer satisfaction, and operational efficiency.

📊  Result of the project
Identified top-selling products and categories contributing the highest revenue
Analyzed monthly and yearly sales trends to understand seasonal demand
Evaluated customer purchase patterns and repeat buying behavior
Highlighted low-performing products and regions requiring improvement
Created an interactive dashboard for quick business insights and decision-making

🛠️ Tools Used
Microsoft Excel (data cleaning, formulas, and conditional formatting), Pivot Tables, Pivot Charts, basic statistical analysis, and business KPI calculations. Skills
Advanced Excel · Pivot Tables · Pivot Charts · VLOOKUP · INDEX & MATCH · Excel Dashboards · Data Analysis · Data Cleaning

💎 Challenges Faced
Handling missing and inconsistent data across multiple datasets
Creating proper relationships between sales, customer, and product tables
Designing meaningful KPIs such as sales growth, profit margin, and customer retention
Optimizing dashboard performance with large datasets
Converting business requirements into clear visual insights. 

Conclusion
This capstone project demonstrates how data analytics can transform raw e-commerce data into actionable business insights.
The dashboard enables stakeholders to track performance, identify growth opportunities, and make informed strategic decisions.
